# Digital Piggy Bank Tracker

A clean, modern desktop companion application built with Python and **CustomTkinter** designed to bridge the gap between physical savings and digital tracking. 

## The Problem & Solution
While saving physical cash in a real-world piggy bank is satisfying, tracking the exact balance over time or after multiple deposits is highly inconvenient. This app serves as a digital ledger—every time you drop money into your physical piggy bank, you log it here to maintain a flawless, real-time visual balance.

## Features
* **Modern UI/UX:** Built using CustomTkinter for a sleek, dark-themed aesthetic.
* **Real-time Ledger:** Easily input deposits and track your cumulative total.
* **Frictionless Actions:** Quick entry fields designed for rapid tracking without complex accounting bloat.

## Tech Stack & Concepts Used
* **Language:** Python
* **GUI Framework:** CustomTkinter (Tkinter extension for modern widgets)
* **Concepts:** State management, event handling, component-based GUI design.
